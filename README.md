# FortiShield
FortiShield is an AI-powered cybersecurity application designed to provide intelligent and proactive malware detection. The system performs real-time monitoring of the Downloads folder using the Watchdog library, automatically scanning newly added files, while also offering a manual scan feature to analyze files from any location. It leverages deep learning techniques, where binary files are converted into image representations and analyzed using a Convolutional Neural Network (CNN) to identify malicious patterns, enabling detection of both known and unknown threats. Upon detection, FortiShield initiates automated response actions such as quarantining the file, generating alerts, and logging threat details, ensuring enhanced system security through continuous monitoring, intelligent analysis, and user-friendly protection.
